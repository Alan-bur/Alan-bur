I'm not just another modern UI/UX Designer. I'm one hopeful person who thinks we should bring the best from 2000s aesthetics back to user interfaces. Nowadays interfaces look boring and lack personality. I don't think discarding [Frutiger Aero](https://en.wikipedia.org/wiki/Frutiger_Aero) entirely was the answer. Why not keeping the best of it? We could have the best user experience and a pretty interface all together!
